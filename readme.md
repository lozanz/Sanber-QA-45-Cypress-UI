# Sanber-QA-45-Cypress-UI
UI Testing Using Cypress

How?
```
npm install cypress
```

in the folder consists of UI automation tests :
- login.e2e.js
- category.e2e.js 
- product.e2e.js 
- customer.e2e.js 

and file runtest.js is test suite from login to customer

How to open & run test : 
```
npx cypress open
npx cypress run
```
